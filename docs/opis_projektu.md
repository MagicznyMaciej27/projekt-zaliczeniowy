# Opis projektu

Projekt dotyczy aplikacji wspierającej kolekcjonerów monet
w katalogowaniu i zarządzaniu swoimi zbiorami.

Użytkownik może dodawać informacje o każdej monecie:
- kraj
- rok wybicia
- nominał
- stan
- własne notatki i opis

Aplikacja pomaga uporządkować zbiory oraz umożliwia
wyszukiwanie i filtrowanie wg własnych parametrów.

Odbiorcami są miłośnicy numizmatyki, którzy lubią historię i porządek.
